cs.status is a script in CIMEROOT/scripts/Tools.

$ ./cs.status --help
usage: 
cs.status <Glob of TestStatus> [<Path/Glob to TestStatus> ...]  [--verbose]
OR
cs.status --help
OR
cs.status --test

[1mEXAMPLES:[0m
    [1;32m# Wait for all tests in a test area[0m
    > cs.status path/to/testarea/*/TestStatus

List test results based on TestStatus files. Returns True if no errors occured
(not based on test statuses).

positional arguments:
  paths                 Paths to TestStatus files. (default: None)

optional arguments:
  -h, --help            show this help message and exit
  -s, --summary         Only show summary (default: False)
  -t TEST_ID, --test-id TEST_ID
                        Only show summary (default: [])
  -r TEST_ROOT, --test-root TEST_ROOT
                        Only show summary (default:
                        /Users/aliceb/cime/scripts/Tools)
