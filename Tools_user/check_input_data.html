check_input_data is a script in CIMEROOT/scripts/Tools.

$ ./check_input_data --help
usage: 
check_input_data [--download] [--verbose]
OR
check_input_data --help
OR
check_input_data --test

[1mEXAMPLES:[0m
    [1;32m# Download input data [0m
    > check_input_data --download

Check (and optionally download) input data files. Should be run from case.

optional arguments:
  -h, --help            show this help message and exit
  -d, --debug           Print debug information (very verbose) to file
                        /Users/aliceb/cime/scripts/Tools/check_input_data.log
                        (default: False)
  -v, --verbose         Add additional context (time and file) to log messages
                        (default: False)
  -s, --silent          Print only warnings and error messages (default:
                        False)
  --svn-loc SVN_LOC     The input data repository from which to download data.
                        (default: https://acme-svn2.ornl.gov/acme-
                        repo/acme/inputdata)
  -i INPUT_DATA_ROOT, --input-data-root INPUT_DATA_ROOT
                        The root directory where input data goes, use xmlquery
                        DIN_LOC_ROOT to see default value. (default: None)
  --data-list-dir DATA_LIST_DIR
                        Where to find list of input files (default: Buildconf)
  --download            Attempt to download missing input files (default:
                        False)
