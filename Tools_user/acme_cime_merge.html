A script to merge ESMCI's cime development and make it into a PR for ACME

positional arguments:
  repo           Location of repo to use, default is based on current location
                 (default: None)

optional arguments:
  -h, --help     show this help message and exit
  -d, --debug    Print debug information (very verbose) to file /Users/aliceb/
                 cesm2_0_beta07/cime/scripts/Tools/acme_cime_merge.log
                 (default: False)
  -v, --verbose  Add additional context (time and file) to log messages
                 (default: False)
  -s, --silent   Print only warnings and error messages (default: False)
  --resume       Resume merge after fixing conflicts (default: False)
