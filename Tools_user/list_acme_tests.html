list_acme_tests is a script in CIMEROOT/scripts/Tools.

$ ./list_acme_tests --help
usage: 
list_acme_tests <thing-to-list> [<test category> <test category> ...] [--verbose]
OR
list_acme_tests --help
OR
list_acme_tests --test

[1mEXAMPLES:[0m
    [1;32m# List all tested compsets [0m
    > list_acme_tests compsets
    [1;32m# List all compsets tested by acme_developer [0m
    > list_acme_tests compsets acme_developer
    [1;32m# List all grids tested by acme_developer [0m
    > list_acme_tests grid acme_developer

List acme test suites. Can be used to show what's being tested. Can just list
tested grids, compsets, etc.

positional arguments:
  {compsets,grids,testcases,tests}
                        The thing you want to list
  categories            The test categories to list. Default will list all.
                        Test categories: acme_runoff_developer,
                        acme_land_developer, cime_tiny, cime_developer,
                        cime_test_only, acme_integration,
                        cime_test_only_slow_pass, acme_developer,
                        cime_test_only_pass, acme_atm_developer (default:
                        None)

optional arguments:
  -h, --help            show this help message and exit
  -d, --debug           Print debug information (very verbose) to file
                        /Users/aliceb/cime/scripts/Tools/list_acme_tests.log
                        (default: False)
  -v, --verbose         Add additional context (time and file) to log messages
                        (default: False)
  -s, --silent          Print only warnings and error messages (default:
                        False)
