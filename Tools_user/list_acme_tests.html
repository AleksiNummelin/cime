list_acme_tests is a script in CIMEROOT/scripts/Tools.

$ ./list_acme_tests --help
usage: 
list_acme_tests <thing-to-list> [<test category> <test category> ...] [--verbose]
OR
list_acme_tests --help

[1mEXAMPLES:[0m
    [1;32m# List all tested compsets [0m
    > list_acme_tests compsets
    [1;32m# List all compsets tested by acme_developer [0m
    > list_acme_tests compsets acme_developer
    [1;32m# List all grids tested by acme_developer [0m
    > list_acme_tests grid acme_developer

List acme test suites. Can be used to show what's being tested. Can just list
tested grids, compsets, etc.

positional arguments:
  suites                The tests suites to list. Test suites: acme_hi_res,
                        acme_runoff_developer, acme_land_developer,
                        acme_atm_extra_coverage, cime_tiny, cime_test_only,
                        acme_integration, acme_developer, acme_atm_developer,
                        fates, acme_atm_prod, acme_extra_coverage,
                        cime_developer, acme_atm_integration,
                        cime_test_only_slow_pass, acme_prod,
                        cime_test_only_pass, acme_test_me, acme_atm_hi_res

optional arguments:
  -h, --help            show this help message and exit
  -d, --debug           Print debug information (very verbose) to file /Users/
                        aliceb/cesm2_0_beta07/cime/scripts/Tools/list_acme_tes
                        ts.log (default: False)
  -v, --verbose         Add additional context (time and file) to log messages
                        (default: False)
  -s, --silent          Print only warnings and error messages (default:
                        False)
  -t {compsets,grids,testcases,tests}, --thing-to-list {compsets,grids,testcases,tests}
                        The thing you want to list (default: tests)
